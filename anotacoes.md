# HTML5 E CSS3

## HTML

### Estrutura básica de um documento HTML:

```
<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset=""UTF-8>
    <title>Nome do documento</title>
  </head>

  <body>

  </body>

</html>
```

`h1` -> Cria um título <br>
`hr` -> Cria uma linha horizontal <br>
`p` -> Cria um parágrafo <br>
`br` -> Quebra um texto <br>
`&lt;` -> Cria um < <br>
`&gt;` -> Cria um > <br>
